# 🔐 Password Generator

A simple web-based password generator that creates strong and secure passwords.

## 🚀 Features
- Generate random passwords
- Customize length
- Include letters, numbers, and symbols
- Copy to clipboard

## 🛠️ Tech Used

- CSS
- JavaScript
